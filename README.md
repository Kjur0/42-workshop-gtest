# `gtest` workshop

## Goal

Learn how to use the `gtest` framework to write unit tests for your C/C++ code.

## How to use

1. Clone the repository
2. Open your code editor and navigate to the `learn` folder
3. Open a given exercise file (e.g., `ex00.cpp`, `ex01.cpp`, `ex02.cpp`, etc.) and edit the code (everything you need is in the comments)
4. Run `./learn.sh`

## Stucture

```
learn/
├── All the exercises are in this folder
res/
├── All the resources needed for the exercises are in this folder (don't edit this folder)
out/
├── All the output files are in this folder (don't edit this folder)
ft-libft/
├── The `libft` library is in this folder (don't edit this folder)
solutions/
├── All the solutions are in this folder (don't edit this folder)
```
